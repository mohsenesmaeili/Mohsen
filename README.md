# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/mohsenesmaeili/Mohsen.git
cd Mohsen
chmod +x mohsen.sh
./mohsen.sh install
./mohsen.sh 
# Enter a phone number & confirmation code.

# For Auto Launch:
cd Mohsen
chmod 777 mohsen.sh
screen ./autobd.sh
```
